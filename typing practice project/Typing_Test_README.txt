
Typing Test With Paragraph Selection
=====================================

This is a simple Typing Test application built using Python and the Tkinter library. It allows users to practice typing by selecting predefined paragraphs, measuring their typing speed, accuracy, and errors.

Features:
---------
1. **Paragraph Selection**: Users can choose from multiple predefined paragraphs for typing practice.
2. **Dynamic Font Scaling**: The application adjusts font sizes based on screen resolution.
3. **Real-Time Error Detection**: Highlights errors in real-time as the user types.
4. **Score Tracking**: Displays scores, missed attempts, errors, and time taken.
5. **Accuracy Measurement**: Provides feedback on typing accuracy.
6. **Fullscreen Mode**: Utilizes the entire screen for an immersive typing experience.
7. **Congratulations Message**: Notifies the user upon successful completion of the paragraph.

How to Run:
-----------
1. Install Python on your system if not already installed.
2. Save the provided script as `typing_test.py`.
3. Run the script using the command: `python typing_test.py`.

Instructions:
-------------
1. Start the application.
2. Select a paragraph from the dropdown menu.
3. Click the "Start Game" button to begin.
4. Type the displayed paragraph in the text entry box.
5. Press `Enter` once the paragraph is completed.
6. Check your results: Score, Time, and Accuracy.

Dependencies:
-------------
- Python (Tested on Python 3.8+)
- Tkinter (Built-in with Python)

File Structure:

- `typing_test.py`: Main script for the application.
